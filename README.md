# 🩺 Diabetes Prediction System

This project uses Machine Learning techniques to predict whether a person is diabetic or not based on medical input data.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

---

## 📂 Files Included

- diabetes_prediction_system.ipynb
- diabetes.csv
- README.md

---

## 🎯 Project Objective

The goal of this project is to build a predictive Machine Learning model that can help identify diabetes risk using healthcare data.

---

## 📊 Features

- Data preprocessing
- Data visualization
- Machine Learning model training
- Prediction system
- Accuracy evaluation
